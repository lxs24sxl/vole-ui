# vole-ui
A Vue.js 2.0 Easy UI Toolkit for Web


## Install
```shell
npm install vole-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import VoleUI from 'vole-ui'

Vue.use(VoleUI)
```
